## 🎓 Career Pathway Platform

##  Overview

The **Career Pathway Platform** is a web application designed to bridge the gap between students and employers by providing a centralized platform for career exploration, internship opportunities, job applications, and professional networking.

The platform enables students to discover career paths, search for internships and jobs, connect with companies, and access career development resources. Companies can post job opportunities, manage applications, and engage with potential candidates.

##  Installation

Clone the repository

```bash
git clone https://github.com/kuntal_1007/career-pathway-platform.git
```

Navigate into the project

```bash
cd career-pathway-platform
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm start
```

The application will be available at:

```text
http://localhost:3000
```
